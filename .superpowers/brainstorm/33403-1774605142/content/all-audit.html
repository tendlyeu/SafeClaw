The "User" column and user filter dropdown only appear for admins. Non-admin users see the existing audit page filtered to their own data.

Design notes:

  • Same page, enhanced — not a separate route. Admin gets a user filter dropdown on the existing /dashboard/audit page
  • Admin filter section — visually separated with a divider and "Admin" label
  • "All users" default — shows combined view. Can filter to a single user
  • User column — added to the table when viewing all users
  • Disabled user — shown with strikethrough username
  • Non-admins — see the exact same page they see today, no changes
