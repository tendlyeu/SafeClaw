env badge = admin set via SAFECLAW_ADMINS env var (cannot be demoted via UI). Your own row has no demote/disable buttons.

Key details in this design:

  • Stats bar — quick count of total, admins, disabled
  • "env" badge — distinguishes env var admins from UI-promoted admins
  • No self-actions — you can't demote or disable yourself
  • Disabled users — shown dimmed with "Enable" button instead of Disable
  • "View →" — drills into user detail (their keys, prefs, audit)
  • Promote/Demote — toggles admin status (hidden for env var admins)
